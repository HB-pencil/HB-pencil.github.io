经验总结（一）

Aug 11, 2019 | 计算机基础Android知识

Reset 单个文件（只能默认mixed）

  • Git reset head~2 /path/file

  • Git commit -m “message”

  • Git checkout /path/file

  • Git reset head /path/file

Dialog.show()

  • 自定义DialogFragment Dialog的宽高默认都是wrap_content -2

  • 加载布局 参数为 false

  • dialog.getWindow().setBackgroundDrawable(new ColorDrawable(Color.TRANSPARENT))必需

  • 如果需要改，需要拿到dialog的window通常在onStart里面改，必须在onCreateDialog之后
